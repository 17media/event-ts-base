## Infrastructure

Everything you needed for typescript project init. it includes:

* TSLint
* `tsconfig.json`
* Prettier with precommit hook integration
* `ts-node` for debugging in `.vscode/launch.json`

## Installation

```sh
yarn add --dev github:17media/ts-base
```

## Initialization

```sh
yarn ts-base-init
```
